


# ⏰ Digital Clock in C

A simple **Digital Clock** built in C that displays the current system time in `HH:MM:SS` format, updating every second.  
This project demonstrates the use of standard C libraries for time handling and terminal output.

---

## 📖 Features
- Displays time in **24-hour format** (`HH:MM:SS`)
- Updates every second in place (no scrolling output)
- Lightweight and runs in the terminal
- Beginner-friendly C project

---

## 🛠️ Technologies Used
- **C Language**
- Standard libraries: `<stdio.h>`, `<time.h>`, `<unistd.h>`

---

## 🚀 How to Run

### 1. Clone the repository
```bash
git clone https://github.com/rohansahu41/DigitalClock.git
cd DigitalClock
```

### 2. Compile the program
On Linux/macOS:
```bash
gcc Clock.c -o Clock
```

On Windows (MinGW or similar):
```bash
gcc Clock.c -o Clock.exe
```

### 3. Run the executable
```bash
./Clock
```
or on Windows:
```bash
Clock.exe
```
